# Codex 快捷指令

這份文件記錄本專案和 Codex 協作時的固定短指令。

## 保養

使用者輸入「保養」時，代表要做專案整理與開發環境維護。

預期處理內容：
- 檢查目前專案狀態與主要檔案。
- 更新狀態、規則、下一步、決策紀錄與已知問題文件。
- 檢查預約、會員、課程、教練、營運摘要等主要流程是否有明顯問題。
- 執行可行的語法檢查與瀏覽器基本檢查。
- 回報目前狀態、已整理項目與下一步建議。

## 上傳

使用者輸入「上傳」時，代表要把最新版本同步到 GitHub Pages 展示版。

預期處理內容：
- 檢查不要上傳真實會員資料、Excel、CSV、log 或私密檔案。
- 執行可行的語法檢查與基本畫面檢查。
- 提交並推送最新版本到 GitHub。
- 若命令列工具不可用，改引導使用者用 GitHub Desktop 完成 Commit / Push。
- 上傳後提醒展示網址：
  `https://tyehyu.github.io/yugym-booking-system-app/`

## 展示版提醒

- GitHub Pages 展示版適合討論流程、畫面與除錯。
- 展示版目前不是正式多人共用資料庫。
- 不同電腦的瀏覽器資料不會自動同步。
- 正式多人共用需要接上 Supabase 或其他後端資料庫。
