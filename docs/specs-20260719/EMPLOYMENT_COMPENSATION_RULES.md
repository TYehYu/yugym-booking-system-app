# YUGYM 員工聘用、薪酬與福利規則

狀態：營運規則草案（已確認部分）  
原則：聘用身份、值班、授課、主管職務、店長職務、保險分開計算，允許個人覆寫並保留生效期間。

## 一、聘用身份模板

| 項目 | 正職 | 兼職 | 合作 | 工讀 |
| --- | ---: | ---: | ---: | ---: |
| 保障底薪 | 29,500 元／月 | 無 | 無 | 無 |
| 值班薪資 | 底薪涵蓋固定值班 | 200 元／小時 | 若有值班則 200 元／小時 | 200 元／小時 |
| 每週值班 | 固定 5 天 | 至少 3 天 | 不值班或個別約定 | 無最低天數，可不值班 |
| 每和值班 | 6 小時 | 6 小時 | 依實際排班 | 有排班才依實際時數 |
| 教練課堂獎金 | 階段制（見下表） | 600 元／堂 | 600 元／堂 | 600 元／堂 |
| 教練課達標獎金 | 80 堂 5,000；100 堂再加 3,000 | 未設定 | 未設定 | 未設定 |
| 團課獎金 | 200 元／實到人數 | 200 元／實到人數 | 200 元／實到人數 | 200 元／實到人數 |
| 勞健保 | 有 | 有 | 無 | 無 |

## 二、職務加給

### 正職教練課堂獎金

當月只統計「教練課」與「友善課」：

| 當月有效堂數 | 堂獎單價 |
| ---: | ---: |
| 1–59 堂 | 600 元／堂 |
| 60–79 堂 | 650 元／堂 |
| 80–99 堂 | 675 元／堂 |
| 100 堂以上 | 700 元／堂 |

達標獎金另外計算：

| 當月有效堂數 | 達標獎金 |
| ---: | ---: |
| 未達 80 堂 | 0 元 |
| 80–99 堂 | 5,000 元 |
| 100 堂以上 | 8,000 元（5,000 + 3,000） |

堂獎採各級距分段累進，不會因達到下一級而回頭調高前面堂數：

- 第 1–59 堂：每堂 600 元。
- 第 60–79 堂：只有這 20 堂每堂 650 元。
- 第 80–99 堂：只有這 20 堂每堂 675 元。
- 第 100 堂起：只有第 100 堂以後每堂 700 元。

範例：當月 85 堂的堂獎為 `59 × 600 + 20 × 650 + 6 × 675 = 52,450 元`，另加 80 堂達標獎金 5,000 元。

### 全體教練續約獎金

- 適用對象：所有教練，不分正職、兼職、合作或工讀。
- 適用方案：教練課及友善課票券。
- 獎金：每張有效續約票券 1,000 元。
- 團課、自主訓練、補課券及其他票券不計續約獎金。
- 預設歸屬會員當時的主教練；櫃台可在交易中改指定其他教練。
- 櫃台覆寫時必須保存原預設教練、指定教練、原因、操作者與時間。
- 付款完成時認列獎金；未完成付款不認列。
- 票券退款或作廢時必須追回。系統新增一筆 `-1,000 元` 的沖回明細並連結原交易，不刪除原獎金紀錄。
- 若已跨月結算，沖回列入退款／作廢發生月份，避免回改已鎖定的舊薪資單。

### 主管

- 主管獎金：4,000 元／月。
- 主管身份與聘用身份分開，可套用在正職、兼職或其他經核准身份。
- 主管統一於每月 1 日生效，不做月中按日比例計算。

### 店長

以「該分店、該薪資結算月份、每一位教練的有效教練課堂數」分別計算：

| 單一教練月堂數 | 店長獎金 |
| ---: | ---: |
| 未達 80 堂 | 0 元 |
| 80–99 堂 | 4,000 元 |
| 100 堂以上 | 6,000 元（4,000 + 2,000） |

- 每位達標教練獨立產生一份獎金，再加總給該分店店長。
- 範例：A 教練 85 堂、B 教練 105 堂，店長當月獎金為 4,000 + 6,000 = 10,000 元。
- 店長獎金可與主管獎金 4,000 元同時領取。

## 三、計薪資料來源

- 值班時數：以核准班表與打卡結果計算；異常補打卡需先核准。
- 教練課堂數：只看課程類型；完成簽到且未取消的「教練課」與「友善課」才計入。
- 體驗課、團課、自主訓練及其他課程類型不計；事後補簽仍只算原本那一堂，不重複增加堂數。
- 團課人數：以每堂實際簽到會員人數計算，不以預約人數計算。
- 分店歸屬：依課程發生時的分店快照計算，員工日後轉店不回改舊薪資。
- 薪資結算後保存計算快照；後續改模板不影響已結算月份。

## 四、系統資料設計

1. `employment_templates`：正職、兼職、合作、工讀的預設條件。
2. `employee_contract_versions`：每位員工的身份、底薪、時薪、生效日與結束日。
3. `employee_duty_rules`：每週最低／固定天數、每日時數、是否必須值班。
4. `employee_compensation_rules`：堂獎、階段制、達標獎金與團課人頭獎金。
5. `employee_benefits`：勞保、健保與其他福利，與登入權限分離。
6. `employee_position_assignments`：主管、店長、所屬分店與生效期間。
7. `payroll_snapshots`：每月各項計算依據、數量、單價與金額快照。

## 五、尚待營運確認

1. 分期付款的「付款完成」是每期款各自完成時認列，或整張票券全額付清才認列。
2. 同一張續約票券只發一份 1,000 元，或允許分配給兩位教練。
3. 合作人員若選擇值班，是否有最低時數。
