# YUGYM OS 上線驗收清單

## 測試前

- 測試專案 migration 與正式專案清單一致。
- 使用匿名、會員、教練、櫃台、管理員五組獨立帳號。
- 測試會員只使用測試票券，不更動正式會員紀錄。

## P0 端到端流程

| 編號 | 角色 | 操作 | 預期結果 |
| --- | --- | --- | --- |
| UAT-01 | 會員 | 建立自己的預約 | 成功並只扣自己的最早到期票券 |
| UAT-02 | 會員 | 嘗試替別人預約 | `AUTH.FORBIDDEN` |
| UAT-03 | 櫃台 | 建立、修改會員預約 | 成功且保留操作紀錄 |
| UAT-04 | 教練 | 簽到自己的課 | 成功且重複操作不重複發點 |
| UAT-05 | 櫃台 | 課程當日簽到 | 發 2 點；課程日第 1 天，第 7 天到期 |
| UAT-06 | 櫃台 | 次日補簽 | 仍發 2 點；期限仍依原課程日計算 |
| UAT-07 | 櫃台 | 24 小時前取消 | 退回原票券批次與原到期日 |
| UAT-08 | 櫃台 | 24 小時內取消 | 不退堂；教練／友善課標記待補課 |
| UAT-09 | 新會員 | 公開申辦 | 建立 Auth 與新 member，不能讀到其他會員 |
| UAT-10 | 舊會員 | 相同手機申辦 | 只建立待審申請，不直接改 `auth_id` |
| UAT-11 | 櫃台 | 核准舊會員連結 | 保留原 member ID、票券、預約與紀錄 |
| UAT-12 | 教練 | 查看非自己會員敏感資料 | 遭 RLS 阻擋 |
| UAT-13 | 會員 | 查看他人票券／預約 | 遭 RLS 阻擋 |
| UAT-14 | 匿名 | 呼叫所有 SECURITY DEFINER RPC | 遭拒絕 |
| UAT-15 | 櫃台 | 首頁點「簽到管理」 | 進入今日會員簽到工作台，不會進入員工打卡 |
| UAT-16 | 櫃台 | 在開課前 30 分鐘內替單人課簽到 | 成功更新狀態；重複操作不重複發點 |
| UAT-17 | 櫃台 | 替團體課簽到 | 開啟名單並可逐一設定到場狀態 |
